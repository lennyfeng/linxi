# Visual Style Proposal — 霖淅内部平台

> Reference: Feidee Cloud (随手记云端) + speckit.specify
> Objective: 清新爽快、数据密集但不压迫、财务专业感

---

## A. 总体风格定位

### 参考图分析（随手记云端）
从截图提取的核心设计语言：

| 特征 | 随手记做法 | 
|------|-----------|
| **色调** | 暖橙色主色调，白色面板，浅灰背景 |
| **布局** | 左侧菜单 + 内容区左侧月份导航 + 右侧流水列表 |
| **数据展示** | 无网格线表格，日期分组，类别图标+彩色圆底 |
| **层次感** | 靠背景色差区分层级，极少投影 |
| **留白** | 充足，每行数据间距宽松 |
| **操作** | 顶部 "+ 记一笔" 醒目按钮，右上角筛选/搜索 |
| **数字排版** | 金额突出、右对齐、收入绿色/支出红色 |

## B. 需要确认的设计方向（5 个关键决策）

### Decision 1: 主色调

| 方案 | 色值 | 感觉 | 适合场景 |
|------|------|------|---------|
| **A. 薄荷绿** | `#00B894` | 清新、专业、安静 | 财务工具、数据密集型 |
| **B. 暖橙色**（随手记同款） | `#FF6B35` | 活力、亲切、温暖 | 个人记账、C端产品 |
| **C. 靛蓝色** | `#4361EE` | 稳重、科技感 | 企业后台、SaaS |

**我的推荐**: **A. 薄荷绿 `#00B894`**
- 理由：内部 B 端工具需要专业感而非亲切感；绿色天然关联"钱/财务"；长时间使用不疲劳；与收支红绿色系搭配协调

---

### Decision 2: 布局结构

| 方案 | 描述 |
|------|------|
| **A. 随手记双面板** | 左侧月份导航 + 右侧日期分组流水（仅 Ledger 模块） |
| **B. 标准 ProTable** | 传统过滤栏 + 分页表格（所有模块统一） |
| **C. 混合** | Ledger 用双面板（像随手记），其他模块用标准 ProTable |

**我的推荐**: **C. 混合**
- Ledger 是财务高频操作页面，双面板的月份导航 + 日期分组更直观
- 其他模块（勾稽、产品开发等）数据结构不同，标准表格更合适

---

### Decision 3: 侧边栏风格

| 方案 | 描述 |
|------|------|
| **A. 白色侧边栏**（随手记同款） | 干净清爽，与内容区靠极细分割线区分 |
| **B. 深色侧边栏** | 沉稳专业，常见于企业后台 |
| **C. 超浅灰侧边栏** | `#FAFBFC` 微灰，比白色稍有层次 |

**我的推荐**: **A. 白色侧边栏**
- 匹配"清新爽快"目标；与随手记一致；简洁不沉重

---

### Decision 4: 表格/列表风格

| 方案 | 描述 |
|------|------|
| **A. 无边框行**（随手记同款） | 行间用极细灰线分隔，无竖线，无表头背景 |
| **B. 斑马纹** | 奇偶行交替浅灰背景 |
| **C. 卡片行** | 每条数据是一个微型卡片，有圆角和微投影 |

**我的推荐**: **A. 无边框行**
- 最干净，数据密集时不杂乱；与随手记参考一致

---

### Decision 5: 分类图标

| 方案 | 描述 |
|------|------|
| **A. 彩色圆底图标**（随手记同款） | 每个分类配一个 Lucide 图标 + 彩色圆形底色 |
| **B. 纯文字标签** | 分类用彩色 Tag 展示，无图标 |
| **C. 单色线条图标** | 图标不上色，统一灰色 |

**我的推荐**: **A. 彩色圆底图标**
- 视觉识别度高；扫一眼列表就能区分不同分类；让页面不单调
